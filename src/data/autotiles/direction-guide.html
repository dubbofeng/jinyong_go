🎮 等距瓦片方向说明

瓦片结构 - 边缘(Edges)

TOP
上边
RIGHT
右边
BOTTOM
下边
LEFT
左边

等距瓦片是菱形，有4条边
每条边可以是不同的地形

瓦片结构 - 角落(Corners)

topLeft
topRight
bottomLeft
bottomRight

4个角落用于精确描述
地形过渡的位置

📍 坐标系统

在笛卡尔坐标系(x, y)中，瓦片的邻居位置：

(x, y-1) ← TOP
(x-1, y) ← LEFT        [当前瓦片]        RIGHT → (x+1, y)
BOTTOM → (x, y+1)

D

C的BOTTOM边 连接 D的TOP边

📊 JSON数据格式

字段 说明 示例
edges.top 上边的地形类型 "water", "sand", "transition"
edges.right 右边的地形类型 "water", "sand", "transition"
edges.bottom 下边的地形类型 "water", "sand", "transition"
edges.left 左边的地形类型 "water", "sand", "transition"

🎯 实际应用

示例：fire-water.png 瓦片[0,0]

{
  "description": "Mostly water with small sand intrusion at top edge",
  "terrain1": "water",
  "terrain2": "sand",
  "edges": {
    "top": "sand",      // ← 上方应该是沙地
    "right": "water",   // ← 右方应该是水域
    "bottom": "water",  // ← 下方应该是水域
    "left": "water"     // ← 左方应该是水域
  }
}

这表示：

  • 这个瓦片的 TOP边 有沙地侵入，所以它上方(y-1)的瓦片应该是沙地或包含沙地
  • 它的 RIGHT/BOTTOM/LEFT边 都是水域，所以那些方向的瓦片应该是水域或包含水域

自动选择瓦片算法

// 1. 检查当前位置(x, y)和邻居的地形
const currentTerrain = map[y][x];
const topTerrain = map[y-1]?.[x];    // 上方瓦片
const rightTerrain = map[y][x+1];    // 右方瓦片
const bottomTerrain = map[y+1]?.[x]; // 下方瓦片
const leftTerrain = map[y][x-1];     // 左方瓦片

// 2. 遍历autotile配置，找到edges匹配的瓦片
for (const tile of autotileConfig.tiles) {
  if (tile.isEmpty) continue;
  
  // 检查边缘是否匹配
  const topMatch = (tile.edges.top === topTerrain || tile.edges.top === 'transition');
  const rightMatch = (tile.edges.right === rightTerrain || tile.edges.right === 'transition');
  const bottomMatch = (tile.edges.bottom === bottomTerrain || tile.edges.bottom === 'transition');
  const leftMatch = (tile.edges.left === leftTerrain || tile.edges.left === 'transition');
  
  if (topMatch && rightMatch && bottomMatch && leftMatch) {
    return tile.index; // ← 找到匹配的瓦片索引
  }
}
