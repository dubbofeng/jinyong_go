# 文档一：游戏策划文档

## 项目概述

### 游戏名称
《金庸棋侠传》

### 游戏类型
武侠题材围棋策略网页游戏

### 核心概念
玩家在现代围棋道场打谱时，意外穿越到金庸武侠世界，发现这个世界中围棋修为决定武功高低。玩家需要通过学习围棋技能（以金庸武侠招式形式呈现）来闯荡江湖，最终成为一代棋侠。

## 游戏核心玩法

### 1. 基本游戏循环
```
大地图探索 → 遇见金庸人物 → 触发剧情对话 → 接受围棋挑战
    ↓
进入围棋对战 → 使用武侠技能辅助 → 赢得比赛
    ↓
获得奖励（新技能/声望/经验） → 解锁新区域 → 继续探索
```

### 2. 围棋对战系统
- **棋盘规格**：19路标准棋盘（专业水平）
- **AI引擎**：KataGo职业级神经网络引擎
- **难度等级**：9个难度等级（Lv.1-9），通过maxVisits参数控制思考深度
  - Lv.1-3：新手至初段（25-75次访问）
  - Lv.4-6：业余2-4段（100-150次访问）
  - Lv.7-9：业余5段至职业初段（175-300次访问）
- **对战模式**：
  - PVE剧情对战（固定金庸人物对手）
  - PVP异步对战（玩家间对战）
  - 训练模式（无限制练习）
- **基础规则**：标准围棋规则，包含吃子、劫争、死活判断

### 3. 武侠技能系统（核心创新）

#### 技能分类与效果

**基础技能（新手期）**
1. **亢龙有悔**（郭靖）
   - 效果：悔棋，撤回上一步
   - 限制：每局限用1-9次（随等级提升）

2. **独孤九剑**（令狐冲）
   - 效果：形势判断，显示当前局势优势方
   - 限制：每局限用1-9次（随等级提升）

3. **腹语传音**（虚竹）
   - 效果：显示AI建议的3个最佳落点
   - 限制：每局限用1-9次（随等级提升）

4. **机关算尽**（黄蓉）
   - 效果：开启变化图试下功能，允许建立并保存多个变化分支进行演练与回放
   - 限制：每局限用2次或冷却10步
**战术技能（中期）**

5. **斗转星移**（慕容复）
   - 效果：将一颗棋子从一个星位移动到另一个星位，或交换两个星位的棋子（仅限星位）
   - 限制：每局限用1次

6. **打狗棒法**（洪七公）
   - 效果：打晕对手，使对方下一手“瞎走”（系统从合法着法中随机或选择次优落点代下）
   - 限制：每局限用2次，冷却20步

7. **吸星大法**（任我行）
   - 效果：吸走一个星位上的棋子（仅限星位）
   - 限制：每局限用1次

8. **乾坤大挪移**（张无忌）
   - 效果：将一颗棋子移到棋盘对称的位置，或者两个对称位置的棋子对掉
   - 限制：每局限用1-9次（随等级提升），冷却20步


9. **一阳指**（一灯大师）
   - 效果：将棋盘划分为两半，指定对方下一手只能落在某一半
   - 限制：每局限用2次，冷却20步

**战略技能（后期）**
10. **六脉神剑**（段誉）
   - 效果：将棋盘分为6等份，指定对方下一手只能走在其中一份（1/6）
   - 限制：每局限用1-9次（随等级提升），冷却20步

11. **左右互搏**（周伯通）
   - 效果：随机出现一次连下两手机会（可连续下两手）
   - 限制：每局在第（11-40手）

12. **黯然销魂掌**（杨过）
   - 效果：当累计提子数达到10颗时，触发一次连下两手（可连续下两手）
   - 限制：每局限触发1次，触发后失效

13. **降龙十八掌**（乔峰）
   - 效果：若一次性提子数量达到15颗（单次吃子≥15），则立即判定为胜利（直接获胜）
   - 限制：天降绝技，仅在满足触发条件时生效


### 4. 成长系统

#### 玩家等级
- 共50级，每级需要经验递增
- 升级奖励：技能使用次数增加、AI难度解锁、新区域开放

#### 属性系统
1. **棋力**：决定基础围棋水平
2. **悟性**：影响技能学习速度
3. **声望**：影响NPC互动和任务触发

#### 技能熟练度
- 每个技能独立熟练度
- 使用次数越多，效果越强
- 满熟练度解锁技能变体

#### 技能限制：
每局限用次数
冷却步数
内力消耗（可恢复资源）

### 5. 任务系统

#### 任务类型
1. **主线任务**：推动剧情发展，解锁核心技能
2. **支线任务**：探索世界，获得额外奖励
3. **日常任务**：每日刷新，维持玩家活跃
4. **挑战任务**：围棋专项挑战（死活题、官子题）

任务结构：

text
任务触发 → 任务目标 → 任务完成 → 奖励发放
    ↑           ↓
NPC对话/   围棋对战/    胜利/完成    经验/技能/
地点触发   解谜挑战     条件达成     道具/解锁

#### 任务结构示例
```
任务：拜师洪七公
├─ 触发条件：到达华山区域，等级≥3
├─ 任务目标：
│   ├─ 与洪七公对话
│   ├─ 完成3局9路围棋练习
│   └─ 在1局中使用悔棋技能反败为胜
└─ 任务奖励：
    ├─ 技能：亢龙有悔（一级）
    ├─ 经验：200
    └─ 声望：+50
```

#### 主线剧情大纲

**序章：道观问棋**
- 穿越初至，在道观中遇见"千变万劫"木桑道长
- 木桑道长"棋不离手，随时要找人和他下棋"，决定传授围棋之道
- 学习围棋基础规则，体验"围棋是易学难精之事，下法规矩，一点就会"
- 完成教学后获得木桑道长的认可：
  "你虽初学，但悟性甚高。这江湖中，围棋修为决定武功高低。你且下山闯荡，
  以棋会友，自能领悟真谛。记住：下围棋最讲究悟性！"

**开篇：初入江湖**
- 离开道观，来到中原大地，发现这个世界中围棋与武功息息相关
- 在华山脚下遇见洪七公，他感叹："老叫花子这些年找不到好对手，闷得慌！"
- 与洪七公对弈后，学习"亢龙有悔"技能，领悟"武功与棋艺，都讲究进退有度"
- 初步在江湖中立足，开始了以棋会友的旅程

**崛起：声名鹊起**
- 游历少林、襄阳，与令狐冲、郭靖切磋棋艺
- 令狐冲独自下棋的寂寞被玩家打破，传授"独孤九剑"
- 郭靖虽然棋力不高，但领悟"侠之大者"的精神，对玩家影响深远
- 受邀参加江南棋会，在桃花岛和梅庄大显身手

**历练：四方游历**
- 在江南与黄药师、黑白子相遇
- 破解黄药师的棋阵机关，学习"机关算尽"
- 与黑白子研究《呕血谱》，理解"这第六十七子，下在此处，确是妙着"
- 西域之行遇见何足道，观摩他"以剑尖在地下画棋盘"自我对弈
- 郭襄的一句"何不径弃中原，反取西域"点醒何足道，也启发玩家
- 与张无忌在光明顶对弈，学习"乾坤大挪移"

**危机：生死棋局**
- 来到大理，遇见段誉、黄眉僧、段延庆等高手
- 见证黄眉僧与段延庆的惊世对局：
  * 两人以内功在青石上刻棋盘，争夺先手
  * 黄眉僧"将自己右足小趾斩了下来"换取先手
  * 段誉"伸指在破嗔掌中连写了七步棋"，算清所有着法
  * 这一局关乎段氏家族兴亡，让玩家深刻理解围棋的分量
- 对抗邪派棋手（段延庆、鸠摩智），维护棋界和平
- 与一灯大师论禅悟棋，学习"一阳指"，领悟"佛门棋禅"

**巅峰：华山论棋**
- 参加华山论棋大会，天下棋手云集
- 与周伯通对弈，学习"左右互搏"
- 破解小龙女守护的古墓棋局机关
- 面对终极挑战：少室山"珍珑棋局"
  * 观摩众高手失败：
    - 范百龄"精研围棋数十年"却"喷出一大口鲜血"
    - 慕容复因"执著权势"而"眼前渐渐模糊，差点拔剑自刎"
    - 段延庆因"心术不正"而"越走越偏，再也难以挽救"
    - 鸠摩智虽有大智慧但"心不在此，用情不够"
  * 领悟虚竹"闭目落子"的真谛：
    "不着意于生死，更不着意于胜败"
    "心平气和之至，方能破此奇局"
  * 成功破解珍珑棋局，领悟围棋最高境界

**结局：棋圣之路**
- 在华山论棋大会上与天下高手切磋
- 争夺"棋圣"称号，无论胜败都已超凡入圣
- 多重结局选择：
  * **归乡**：带着所学返回现代，推广围棋文化
    "我在那个世界学会了围棋的真谛，不在胜负，而在心性。"
  * **留世**：留在武侠世界，成为新一代棋道宗师
    "这个世界需要我，我愿意留下传承棋道。"
  * **隐退**：归隐山林，与木桑道长一样逍遥自在
    "胜负已不重要，我只想找个清静之地，棋不离手，自在下棋。"

### 7. NPC系统

#### 金庸人物设计

**序章人物**
- **木桑道长**（"千变万劫"）
  * 年长道士，围棋教习，对围棋痴迷至深
  * 性格特点："棋不离手，随时要找人和他下棋"，"一面下，一面给你解释，同时不住口的吹嘘自己"
  * 教学风格：耐心细致，喜欢自吹自擂但心地善良
  * 重要台词："下围棋最讲究悟性，常言道'二十岁不成国手，终身无望'。"
  * 游戏功能：负责新手教学（围棋基本规则、气与提子、劫争、终局判断与计分）
  * 历史出处：《碧血剑》，金庸笔下最爱棋的人物之一

**第一章人物（1-15级）**
- **洪七公**
  * 丐帮帮主，豪爽大气，寻找棋友
  * 性格特点：武功盖世但对围棋也有研究，重情重义
  * 教学内容：传授"亢龙有悔"技能（悔棋）
  * 对战风格：攻守兼备，重视实地
  * 重要台词："老叫花子这些年找不到好对手，闷得慌！"

- **令狐冲**
  * 华山派弟子，剑术超群，性格洒脱
  * 性格特点：独自抚琴下棋，"寂寞之极"，寻找知音
  * 教学内容：传授"独孤九剑"技能（形势判断）
  * 对战风格：灵活多变，善于寻找破绽
  * 重要台词："世间难得知己，你我以棋会友，岂不快哉？"

- **郭靖**
  * 大侠，侠之大者，虽然棋力不高但心性纯正
  * 性格特点：憨厚朴实，领悟"侠之大者，为国为民"
  * 教学内容：不传授技能，但传递武学精神
  * 对战风格：稳扎稳打，不取巧不取巧
  * 重要台词："我虽不擅围棋，但懂得一个道理：无论武功还是棋艺，关键在于心正。"

**第二章人物（16-25级）**
- **黄药师**（东邪）
  * 桃花岛岛主，精通奇门遁甲，以围棋设阵
  * 性格特点：狂傲不羁，智谋过人
  * 教学内容：传授"机关算尽"技能（变化图试下）
  * 对战风格：诡谲多变，棋盘即阵法
  * 重要台词："棋盘就是阵法，每一步都关乎生死。你若能破我这局，自然能得妙法。"
  * 特殊挑战：需要破解他设置的棋阵机关才能过关

- **黑白子**
  * 梅庄二庄主，对围棋痴迷至深
  * 性格特点：淡泊名利，只爱围棋，珍藏《呕血谱》
  * 教学内容：与玩家共同研究古谱，提升棋力
  * 对战风格：师法古人，重视定式和棋理
  * 重要台词：
    - "这一子吗？断又不妥，连也不对，冲是冲不出，做活却又活不成..."
    - "好，这一子下在此处，确是妙着！"（破解第67手时）
  * 历史出处：《笑傲江湖》，向问天用《呕血谱》引他上钩
  * 背景故事：传说国手刘仲甫在骊山败于老媪，"呕血数升"，留下此谱

**第三章人物（26-35级）**
- **何足道**（"琴棋剑三圣"）
  * 昆仑派高手，号称三圣但实为"棋痴"
  * 性格特点：爱棋成痴，"自己跟自己下"，下棋没有对手
  * 教学内容：展示"当局者迷，旁观者清"的棋理
  * 对战风格：布局精妙，但过于执着于计算
  * 重要场景：
    - 玩家发现他"以剑尖在地下画了一张纵横各十九道的棋盘"
    - "自北而南，逐步争到了中原腹地"，自己和自己下到第93手
    - 陷入连环劫，白势"岌岌可危"
  * 历史出处：《倚天屠龙记》，郭襄点醒他"何不径弃中原，反取西域"

- **张无忌**
  * 明教教主，修习"乾坤大挪移"
  * 性格特点：心地善良，天资聪颖
  * 教学内容：传授"乾坤大挪移"技能（棋子对称交换）
  * 对战风格：变化莫测，善于转圜
  * 重要台词："乾坤大挪移不仅是武功，也是棋理。棋盘对称，阴阳调和。"

**第四章人物（36-45级）**
- **段誉**
  * 大理世子，"自幼善弈"，棋艺精湛
  * 性格特点：聪明善良，才气斐然，是围棋天才
  * 教学内容：传授"六脉神剑"技能（棋盘分割）
  * 对战风格：棋思精密，但"爱心太重，不肯弃子"
  * 重要场景：
    - 在万劫谷中为黄眉僧"在其掌中连写了七步棋"
    - 能"算清了段延庆此前所有着法"
  * 历史出处：《天龙八部》第八章《虎啸龙吟》

- **黄眉僧**
  * 大理皇家高僧，为百姓请命
  * 性格特点：棋力一般但意志坚定，为大义可自残
  * 重要场景：与段延庆的惊世对局
    - 以内功在青石上刻棋盘："槌杖越划越慢，不愿自己所刻直线有何深浅不同"
    - 为争先手"将自己右足小趾斩了下来"
    - 说："老僧今年六十九岁，到得七十岁时，我的足趾是奇数"
  * 历史出处：《天龙八部》，此局事关段氏家族兴亡

- **段延庆**（"天下第一恶人"）
  * 前大理太子，如今恶贯满盈
  * 性格特点："不骄不躁，阴沉之极"，棋力高强
  * 对战风格：稳重严谨，步步为营
  * 重要台词："旁观不语真君子，自作主张大丈夫"
  * 历史出处：与黄眉僧对弈，"号称'天下第一恶人'，什么凶残毒辣的事没干过见过"

- **一灯大师**
  * 前大理保定帝，出家为僧
  * 性格特点：佛法精深，棋禅合一
  * 教学内容：传授"一阳指"技能（限制对手落子区域）
  * 对战风格：禅意深远，不拘泥于胜负
  * 重要台词："围棋如人生，执着于胜负便落了下乘。"

**第五章人物（46-50级）**
- **周伯通**（老顽童）
  * 全真派高手，天真烂漫
  * 性格特点：顽皮好动，但武学天赋极高
  * 教学内容：传授"左右互搏"技能（连下两手）
  * 对战风格：出人意料，常有奇思妙想
  * 重要台词："左右互搏之术，不仅能用于武功，用在下棋上也妙不可言！"

- **小龙女**
  * 古墓派传人，清冷孤傲
  * 性格特点：性格冷淡，但心地善良
  * 教学内容：守护古墓机关，棋局即阵法
  * 对战风格：精密细腻，步步为营
  * 重要台词："这古墓中的每一局棋，都是祖师婆婆留下的。你若能破解，便是有缘人。"

- **虚竹**（破解珍珑棋局之人）
  * 少林小和尚，武功低微，棋艺更低
  * 性格特点：内心纯良，"竹节一般的中空、内虚"
  * 关键事件：
    - "闭着眼睛下了一子"破解珍珑棋局
    - 这一着"大违根本棋理"，等于"提剑自刎、横刀自杀"
    - 但正因"不着意于生死，更不着意于胜败"，反而破局
  * 重要启示："心平气和之至，方能破此奇局"
  * 历史出处：《天龙八部》，因破解珍珑棋局获得逍遥子70余年功力

- **慕容复**（珍珑棋局失败者之一）
  * 姑苏慕容氏传人，执着于复国大业
  * 失败原因："执著权势，勇于弃子，却说什么也不肯失势"
  * 失败场景："眼前渐渐模糊，竟拔剑便往颈中刎去"
  * 教训：过于执着于得失，反而迷失本心

- **范百龄**（珍珑棋局失败者之一）
  * 围棋大师，"精研围棋数十年"
  * 失败原因：技术派，只见树木不见森林，无大局观
  * 失败场景："竟喷出一大口鲜血"
  * 教训：过于精细的计算反而束缚了思维

- **鸠摩智**（珍珑棋局失败者之一）
  * 吐蕃国师，武功高强
  * 失败原因："心不在此，用情不够，非此道中人"
  * 教训：虽有大智慧但不够专注

### 10. 新手引导设计

#### 四阶段引导
1. **完全引导期**（1-3级）：棋魂附身，代下教学
2. **半引导期**（4-10级）：实时提示，有限帮助
3. **辅助期**（11-20级）：仅关键提示
4. **独立期**（21+级）：完全自主，开启高手模式

#### 自适应难度
- AI难度随玩家实力动态调整
- 失败过多时提供额外提示
- 连胜时增加挑战难度
