# 文档一：游戏策划文档

## 项目概述

### 游戏名称
《金庸棋侠传》

### 游戏类型
武侠题材围棋策略网页游戏

### 核心概念
玩家在现代围棋道场打谱时，意外穿越到金庸武侠世界，发现这个世界中围棋修为决定武功高低。玩家需要通过学习围棋技能（以金庸武侠招式形式呈现）来闯荡江湖，最终成为一代棋侠。

## 游戏核心玩法

### 1. 基本游戏循环
```
大地图探索 → 遇见金庸人物 → 触发剧情对话 → 接受围棋挑战
    ↓
进入围棋对战 → 使用武侠技能辅助 → 赢得比赛
    ↓
获得奖励（新技能/声望/经验） → 解锁新区域 → 继续探索
```

### 2. 围棋对战系统
- **棋盘规格**：9路（新手）、13路（进阶）、19路（高手）
- **对战模式**：
  - PVE剧情对战（固定金庸人物对手）
  - PVP异步对战（玩家间对战）
  - 训练模式（无限制练习）
- **基础规则**：标准围棋规则，包含吃子、劫争、死活判断

### 3. 武侠技能系统（核心创新）

#### 技能分类与效果

**基础技能（新手期）**
1. **亢龙有悔**（郭靖）
   - 效果：悔棋，撤回上一步
   - 限制：每局限用1-9次（随等级提升）

2. **独孤九剑**（令狐冲）
   - 效果：形势判断，显示当前局势优势方
   - 限制：每局限用1-9次（随等级提升）

3. **腹语传音**（虚竹）
   - 效果：显示AI建议的3个最佳落点
   - 限制：每局限用1-9次（随等级提升）

4. **神机妙算**（黄蓉）
   - 效果：开启变化图试下功能，允许建立并保存多个变化分支进行演练与回放
   - 限制：每局限用2次或冷却10步
**战术技能（中期）**

5. **斗转星移**（慕容复）
   - 效果：将一颗棋子从一个星位移动到另一个星位，或交换两个星位的棋子（仅限星位）
   - 限制：每局限用1次

6. **打狗棒法**（洪七公）
   - 效果：打晕对手，使对方下一手“瞎走”（系统从合法着法中随机或选择次优落点代下）
   - 限制：每局限用2次，冷却20步

7. **吸星大法**（任我行）
   - 效果：吸走一个星位上的棋子（仅限星位）
   - 限制：每局限用1次

8. **乾坤大挪移**（张无忌）
   - 效果：将一颗棋子移到棋盘对称的位置，或者两个对称位置的棋子对掉
   - 限制：每局限用1-9次（随等级提升），冷却20步


9. **一阳指**（一灯大师）
   - 效果：将棋盘划分为两半，指定对方下一手只能落在某一半
   - 限制：每局限用2次，冷却20步

**战略技能（后期）**
10. **六脉神剑**（段誉）
   - 效果：将棋盘分为6等份，指定对方下一手只能走在其中一份（1/6）
   - 限制：每局限用1-9次（随等级提升），冷却20步

11. **左右互搏**（周伯通）
   - 效果：随机出现一次连下两手机会（可连续下两手）
   - 限制：每局在第（11-40手）

12. **黯然销魂掌**（杨过）
   - 效果：当累计提子数达到10颗时，触发一次连下两手（可连续下两手）
   - 限制：每局限触发1次，触发后失效

13. **降龙十八掌**（乔峰）
   - 效果：若一次性提子数量达到15颗（单次吃子≥15），则立即判定为胜利（直接获胜）
   - 限制：天降绝技，仅在满足触发条件时生效


### 4. 成长系统

#### 玩家等级
- 共50级，每级需要经验递增
- 升级奖励：技能使用次数增加、新棋盘解锁、新区域开放

#### 属性系统
1. **棋力**：决定基础围棋水平
2. **悟性**：影响技能学习速度
3. **声望**：影响NPC互动和任务触发

#### 技能熟练度
- 每个技能独立熟练度
- 使用次数越多，效果越强
- 满熟练度解锁技能变体

#### 技能限制：
每局限用次数
冷却步数
内力消耗（可恢复资源）

### 5. 任务系统

#### 任务类型
1. **主线任务**：推动剧情发展，解锁核心技能
2. **支线任务**：探索世界，获得额外奖励
3. **日常任务**：每日刷新，维持玩家活跃
4. **挑战任务**：围棋专项挑战（死活题、官子题）

任务结构：

text
任务触发 → 任务目标 → 任务完成 → 奖励发放
    ↑           ↓
NPC对话/   围棋对战/    胜利/完成    经验/技能/
地点触发   解谜挑战     条件达成     道具/解锁

#### 任务结构示例
```
任务：拜师洪七公
├─ 触发条件：到达华山区域，等级≥3
├─ 任务目标：
│   ├─ 与洪七公对话
│   ├─ 完成3局9路围棋练习
│   └─ 在1局中使用悔棋技能反败为胜
└─ 任务奖励：
    ├─ 技能：亢龙有悔（一级）
    ├─ 经验：200
    └─ 声望：+50
```

### 6. 世界与剧情

#### 世界地图结构
```
五大区域（循序渐进解锁）
├─ 第一章：中原风云（1-15级）
│   ├─ 地点：华山、少林寺、襄阳城
│   └─ 关键人物：洪七公、令狐冲、郭靖
├─ 第二章：江南棋会（16-25级）
│   ├─ 地点：桃花岛、梅庄
│   └─ 关键人物：黄药师、黑白子
├─ 第三章：西域争锋（26-35级）
│   ├─ 地点：昆仑山、光明顶
│   └─ 关键人物：何足道、张无忌
├─ 第四章：大理佛缘（36-45级）
│   ├─ 地点：天龙寺、无量山
│   └─ 关键人物：段誉、一灯大师
└─ 第五章：华山论棋（46-50级）
    ├─ 地点：终南山、古墓
    └─ 关键人物：周伯通、小龙女
```

#### 主线剧情大纲
- **开篇**：现代围棋少年穿越，获得基础围棋知识
- **崛起**：拜师学艺，参加地区围棋大赛
- **历练**：游历四方，挑战各路棋手
- **危机**：对抗邪派棋手，维护棋界和平
- **巅峰**：参加华山论棋，争夺棋圣称号
- **结局**：多重结局选择（归乡/留世/隐退）

### 7. NPC系统

#### 金庸人物设计
- **木桑道长**：年长道士，围棋教习，教授围棋基本规则（棋盘、棋子、落子规则、气与提子、劫争、终局判断与计分），负责新手教学与规则引导
- **洪七公**：新手导师，教授基础技能
- **黄药师**：解谜大师，设计复杂棋局挑战
- **周伯通**：趣味角色，教授特殊技能
- **任我行**：反派挑战，考验玩家实力
- **小龙女**：隐藏角色，教授高级技巧

#### NPC交互机制
- **对话系统**：分支对话，影响关系发展
- **好感度系统**：通过完成任务和对弈提升
- **师徒系统**：与特定NPC建立师徒关系

### 8. 社交系统

#### 基础社交功能
1. **好友系统**：添加好友，查看在线状态
2. **观战系统**：观看高手对局，学习技巧
3. **棋谱分享**：生成并分享精彩对局
4. **排行榜**：等级榜、胜率榜、技能榜

#### 社区功能
- **棋友圈**：分享心得，讨论棋局
- **师徒系统**：老手带新手，获得奖励
- **棋社系统**：创建或加入棋社，参加社团活动

### 9. 经济系统

#### 游戏内货币
1. **纹枰银**：基础货币，通过对战获得
2. **棋魂玉**：高级货币，完成任务获得

#### 消耗与获取
- **技能升级**：消耗棋魂玉提升技能等级
- **外观道具**：棋盘皮肤、棋子样式、角色服装
- **功能道具**：双倍经验卡、技能冷却重置

### 10. 新手引导设计

#### 四阶段引导
1. **完全引导期**（1-3级）：棋魂附身，代下教学
2. **半引导期**（4-10级）：实时提示，有限帮助
3. **辅助期**（11-20级）：仅关键提示
4. **独立期**（21+级）：完全自主，开启高手模式

#### 自适应难度
- AI难度随玩家实力动态调整
- 失败过多时提供额外提示
- 连胜时增加挑战难度

### 11. 视觉与音效设计

#### 美术风格
- **棋境特效**：优势时棋盘明亮，劣势时棋盘昏暗
- **技能特效**：每个技能有独特视觉表现
- **角色立绘**：金庸人物水墨风格立绘
- **场景设计**：中国风武侠场景

#### 音效设计
- **环境音效**：不同场景不同背景音
- **棋子音效**：落子、提子、技能触发音
- **角色语音**：关键NPC有配音

### 12. 商业化设计（可选）

#### 免费内容
- 完整主线剧情
- 所有基础技能
- 日常PVE对战

#### 付费点设计
1. **外观道具**：不影响平衡的装饰品
2. **技能皮肤**：改变技能特效
3. **便利功能**：额外存档位、快速匹配
4. **扩展内容**：新剧情章节、新技能

### 13. 运营计划

#### 版本规划
- **V1.0**：核心玩法，前两章内容，基础技能
- **V1.5**：社交系统，PVP对战，第三章内容
- **V2.0**：完整五章，全部技能，移动端优化

#### 更新节奏
- 每月小更新：平衡调整，bug修复
- 每季度大更新：新剧情，新技能
- 年度资料片：大型扩展内容

### 14. 目标用户与市场定位

#### 目标用户
1. **围棋爱好者**（核心用户）：30-50岁，有围棋基础
2. **金庸武侠迷**（重要用户）：25-45岁，喜欢武侠文化
3. **策略游戏玩家**（扩展用户）：20-35岁，喜欢轻度策略

#### 市场定位
- **差异化优势**：围棋+武侠的创新结合
- **学习价值**：在游戏中学习围棋知识
- **文化价值**：传播中国围棋和武侠文化
