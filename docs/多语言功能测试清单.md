## 测试步骤示例

### 完整流程测试
1. 打开浏览器，访问 `http://localhost:9999`
2. 确认首页为中文
3. 点击右上角语言切换器，选择"English"
4. 确认URL变为 `/en/`，内容变为英文
5. 点击"Login"，进入登录页面
6. 登录后自动跳转到游戏页面
7. 确认游戏页面显示英文UI
8. 移动角色到洪七公附近，按空格交互
9. 确认对话显示英文
10. 选择对话选项，确认对话树正确跳转
11. 关闭对话，切换语言回中文
12. 再次与洪七公对话，确认显示中文
13. 通过传送门切换到其他地图
14. 与令狐冲和郭靖交互，测试所有对话节点

## 自动化测试建议

可以编写E2E测试覆盖以下场景：
```typescript
// 示例测试用例
describe('多语言功能', () => {
  it('应该正确加载中文对话', async () => {
    await page.goto('http://localhost:9999/zh/game');
    await interactWithNPC('hong_qigong');
    expect(await getDialogueText()).toContain('洪七公');
  });

  it('应该正确加载英文对话', async () => {
    await page.goto('http://localhost:9999/en/game');
    await interactWithNPC('hong_qigong');
    expect(await getDialogueText()).toContain('Hong Qigong');
  });

  it('语言切换应该更新URL和内容', async () => {
    await page.goto('http://localhost:9999/zh/game');
    await switchLanguage('en');
    expect(page.url()).toContain('/en/game');
    expect(await getPageTitle()).toBe("Jin Yong's Go");
  });
});
```
