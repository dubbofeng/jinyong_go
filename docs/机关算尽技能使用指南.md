# 机关算尽技能使用指南

## 技能概述

**技能名称**：机关算尽  
**关联NPC**：黄蓉  
**学习方式**：完成任务 `quest_005_huang_rong`（击败黄蓉）  
**技能ID**：`jiguan_suanjin`（数据库）/ `jiguansuanjin`（代码）

## 功能说明

"机关算尽"是黄蓉独创的棋法，允许玩家在对局中打开一个**试下棋盘**，在副本中尝试不同的走法，而不会影响实际对局。

### 核心特性

1. **试下棋盘**
   - 弹出独立的Modal窗口
   - 自动复制当前战局的所有棋子
   - 保持当前的下棋顺序（黑/白）

2. **自由试下**
   - 在试下棋盘上可以自由落子
   - 黑白交替，遵循围棋规则
   - 支持提子、打劫等所有规则

3. **试下操作**
   - ⏪ **悔棋**：撤销试下中的最后一步
   - 🔄 **重置**：恢复到初始战局
   - ✅ **完成试下**：关闭试下棋盘，返回实战

4. **使用限制**
   - 每局对弈可使用 **2次**
   - 使用后进入 **10手冷却**
   - 冷却期间按钮置灰，显示剩余冷却手数

## 使用流程

### 1. 学习技能

```
玩家 → 黄蓉对话 → 选择"我想学习机关算尽" → 对弈并击败黄蓉 → 获得技能
```

### 2. 使用技能

1. **对弈中**：点击技能栏的 🧩 **机关算尽** 按钮
2. **试下棋盘**自动打开，显示当前战局
3. 在试下棋盘上**自由尝试各种走法**
4. 可以随时**悔棋**或**重置**到初始状态
5. 试下完成后，点击 **✅ 完成试下** 返回实战
6. 实战棋盘**不受影响**，继续从之前的状态下棋

### 3. 技能状态显示

- **可用**：蓝色渐变按钮，显示"剩余: 2/2"
- **冷却中**：灰色按钮，显示"冷却: X手"
- **用完**：灰色按钮，显示"剩余: 0/2"

## 注意事项

1. **独立实例**
   - 试下棋盘使用独立的 `GoBoard` 和 `GoEngine` 实例
   - 与实战棋盘完全隔离，互不影响

2. **冷却机制**
   - 冷却在**每次落子后**自动减1
   - 黑方落子和白方落子都会更新冷却
   - 悔棋不影响冷却进度

3. **关闭方式**
   - 点击 ✕ 按钮
   - 点击"完成试下"按钮
   - 按 ESC 键
   - 点击背景遮罩

4. **性能优化**
   - Modal使用懒加载，只在打开时渲染
   - 关闭后自动销毁Canvas事件监听器
   - 使用CSS动画实现平滑过渡

## 常见问题

### Q1: 试下棋盘上的操作会影响实战吗？
**A**: 不会。试下棋盘是完全独立的副本，所有操作都不会影响实战棋盘。

### Q2: 可以在试下棋盘上使用其他技能吗？
**A**: 不可以。试下棋盘只用于尝试走法，不支持使用其他技能。

### Q3: 冷却是如何计算的？
**A**: 使用技能后立即开始10手冷却。每走一步（黑或白）冷却减1，直到为0时可以再次使用。

### Q4: 如果我在试下棋盘上走了很多步，关闭后会怎样？
**A**: 关闭试下棋盘后，所有试下操作都会被丢弃，实战棋盘保持原样。

### Q5: 技能使用次数如何恢复？
**A**: 每开始一局新对弈时，技能使用次数会自动重置为2次。
