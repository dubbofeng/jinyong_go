# 技能点分配系统使用指南

## 系统概述

技能点分配系统是一个经典的RPG成长机制。玩家每升一级获得1个技能点，可以自由分配给已解锁的技能。技能等级直接决定使用次数，最高可升到9级。

## 核心规则

### 技能点获取
- **每升1级** = 获得 **1个技能点**
- 技能点累积，不会过期
- 可以自由选择分配给哪个技能

### 技能等级系统
- 最低等级：**Lv.1**（初始解锁时）
- 最高等级：**Lv.9**（满级）
- **等级 = 使用次数**（Lv.5 = 每局可用5次）

### 升级规则
- 消耗 **1技能点** 升1级
- 升级后立即生效（下次对局开始时）
- 满级后无法继续升级

## 技能使用次数

**简化规则：等级 = 使用次数**

所有技能遵循统一规则：
- **Lv.1**: 1次 / 每局
- **Lv.2**: 2次 / 每局
- **Lv.3**: 3次 / 每局
- **Lv.4**: 4次 / 每局
- **Lv.5**: 5次 / 每局
- **Lv.6**: 6次 / 每局
- **Lv.7**: 7次 / 每局
- **Lv.8**: 8次 / 每局
- **Lv.9**: 9次 / 每局（满级）

### 🐉 亢龙有悔（郭靖）
悔棋技能，等级 = 悔棋次数

### ⚔️ 独孤九剑（令狐冲）
形势判断技能，等级 = 判断次数

### 🗨️ 腹语传音（虚竹）
AI建议技能，等级 = 建议次数

### 🧩 机关算尽（黄蓉）
试下棋盘技能，等级 = 试下次数（冷却10手）

## 技能点分配界面

### 打开方式
- 点击游戏主界面的"技能点"按钮
- 快捷键：K（未来）
- 升级时自动弹出提示

### 界面功能

#### 顶部信息
- 玩家当前等级
- 可用技能点数量
- 升级提示说明

#### 技能列表
每个技能显示：
- 技能图标和名称
- 关联的武侠人物
- 当前等级和进度条
- 使用次数说明
- 升级按钮

#### 操作按钮
- **升级** - 消耗1技能点，技能等级+1
- **关闭** - 保存并退出界面

## 未来扩展

### 1. 技能特效升级
- 高等级技能附加特殊效果
- 例如：Lv.5机关算尽可以保存多个变化图

### 2. 技能树系统
- 每个技能可以选择不同的升级路径
- 路径A：增加使用次数
- 路径B：减少冷却时间
- 路径C：增强效果

### 3. 技能重置
- 允许玩家重置技能等级
- 返还一定比例的经验
- 消耗特殊道具

### 4. 技能组合
- 高等级后解锁技能组合
- 例如：同时使用"独孤九剑"+"腹语传音"

### 5. 经验加成
- VIP会员获技能点可以重置吗？
**A**: 目前不支持重置。技能点一旦分配就无法撤回，请谨慎分配。

### Q3: 满级后还能获得技能点吗？
**A**: 可以。继续升级依然会获得技能点，可以分配给其他技能。

### Q4: 可以不分配技能点吗？
**A**: 可以。技能点会一直保留，你可以选择何时分配。

### Q5: 技能等级和玩家等级有什么关系？
**A**: 玩家等级决定获得的技能点数量（每级1点），技能等级决定技能使用次数。两者独立计算。

### Q6: 所有技能都能升到9级吗？
**A**: 是的。所有已解锁的技能都可以升到最高9级如有250经验，升级需要200，升级后会剩余50经验。

### Q3: 满级后还会获得经验吗？
**A**: 会。满级技能依然会获得经验，但无法继续升级。未来可能会添加"技能精通"系统使用这些经验。

### Q4: 可以降级技能吗？
**A**: 目前不支持降级。未来可能会添加"技能重置"功能。

### Q5: 不同技能的经验是分开计算的吗？
**A**: 是的。每个技能都有独立的经验值和等级，互不影响。
