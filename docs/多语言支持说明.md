# 多语言支持说明

## 实现的功能

本游戏现在支持中文（zh）和英文（en）两种语言，包括：

1. **UI界面翻译**：所有按钮、提示、说明文本
2. **NPC对话翻译**：三个NPC（洪七公、令狐冲、郭靖）的完整对话树
3. **动态语言切换**：游戏右上角可随时切换语言

## 文件结构

```
/messages
  ├── zh.json          # 中文翻译
  └── en.json          # 英文翻译

/src/data/dialogues
  ├── hong_qigong.zh.json      # 洪七公中文对话
  ├── hong_qigong.en.json      # 洪七公英文对话
  ├── linghu_chong.zh.json     # 令狐冲中文对话
  ├── linghu_chong.en.json     # 令狐冲英文对话
  ├── guo_jing.zh.json         # 郭靖中文对话
  └── guo_jing.en.json         # 郭靖英文对话

/src/i18n
  └── request.ts       # i18n配置

/app/[locale]          # 带语言路由的页面
  ├── layout.tsx       # 带翻译Provider的布局
  ├── page.tsx         # 首页
  ├── game/            # 游戏页面
  ├── login/           # 登录页面
  ├── register/        # 注册页面
  └── protected/       # 受保护页面
```

## 如何使用

### 访问不同语言版本

- 中文（默认）：`http://localhost:9999/game` 或 `http://localhost:9999/zh/game`
- 英文：`http://localhost:9999/en/game`

### 在组件中使用翻译

#### 服务端组件

```tsx
import { getTranslations } from 'next-intl/server';

export default async function MyPage() {
  const t = await getTranslations('game');
  
  return <h1>{t('title')}</h1>;
}
```

#### 客户端组件

```tsx
'use client';

import { useTranslations } from 'next-intl';

export default function MyComponent() {
  const t = useTranslations('game.dialogue');
  
  return <button>{t('continue')}</button>;
}
```

### 添加新的NPC对话

1. 创建两个JSON文件：
   - `/src/data/dialogues/npc_id.zh.json`
   - `/src/data/dialogues/npc_id.en.json`

2. 对话文件格式：
```json
{
  "npcId": "npc_id",
  "npcName": "NPC名称",
  "startNodeId": "greeting",
  "nodes": [
    {
      "id": "greeting",
      "speaker": "NPC名称",
      "text": "对话内容",
      "options": [
        {
          "text": "选项文本",
          "nextNodeId": "next_node"
        }
      ]
    }
  ]
}
```

3. 对话会根据当前语言自动加载对应版本

### 添加新的UI翻译

在 `messages/zh.json` 和 `messages/en.json` 中添加翻译键：

```json
{
  "game": {
    "newKey": "中文内容"
  }
}
```

## 翻译范围

### 已翻译的内容

#### UI界面 (`messages/*.json`)
- 游戏标题和描述
- 控制说明
- 对话框按钮和提示
- 认证表单（登录/注册）
- 通用文本（欢迎、等级、技能等）

#### NPC对话
- **洪七公**：10个对话节点
  - 教授"亢龙有悔"技能
  - 介绍乔峰对弈
  - 解释"北丐"称号
  
- **令狐冲**：13个对话节点
  - 教授"独孤九剑"技能
  - 讨论剑法与围棋策略
  - 探讨围棋定式

- **郭靖**：15个对话节点
  - 教授"腹语传音"技能
  - 设置等级条件（3级、5级）
  - 包含战斗挑战选项

## 语言切换器

位于游戏界面右上角，提供下拉选择：
- 🇨🇳 中文
- 🇬🇧 English

切换语言后，页面会重新加载以应用新语言的内容。

## 技术实现

### 使用的库
- `next-intl 4.7.0`：Next.js 14 App Router专用国际化库

### 核心特性
1. **URL路由支持**：通过 `[locale]` 动态路由实现
2. **自动回退**：如果指定语言文件不存在，自动回退到中文
3. **服务端渲染**：翻译在服务端完成，SEO友好
4. **类型安全**：TypeScript支持，编译时检查翻译键

### 中间件配置

`middleware.ts` 配置了：
- 支持的语言列表：`['zh', 'en']`
- 默认语言：`zh`
- URL前缀策略：`as-needed`（默认语言不显示前缀）

## 开发建议

1. **保持翻译文件同步**：添加新功能时，同时更新 `zh.json` 和 `en.json`
2. **使用命名空间**：按功能模块组织翻译键（如 `game.`, `auth.`, `common.`）
3. **避免硬编码**：所有用户可见的文本都应该通过翻译系统
4. **测试两种语言**：确保UI在不同语言下显示正常
5. **文化适配**：注意文化差异，不仅是语言翻译（如日期格式、货币等）

## 未来扩展

可以轻松添加更多语言：
1. 在 `src/i18n/request.ts` 中添加语言代码到 `locales` 数组
2. 创建对应的 `messages/{locale}.json`
3. 为每个NPC创建对应的对话文件
4. 在 `LanguageSwitcher.tsx` 中添加语言选项
