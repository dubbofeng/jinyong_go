# MVP开发计划文档

## 概述

**MVP目标**：快速验证游戏核心玩法（围棋对战 + 武侠技能系统），建立可玩的基础游戏体验。

**时间预估**：4-6周
**目标范围**：第一章内容（华山、少林寺、襄阳城）
**核心功能**：5个基础技能、单机剧情、围棋对战
