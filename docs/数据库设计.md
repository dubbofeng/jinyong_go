## 概述

本文档描述金庸围棋游戏的完整数据库设计，包括用户系统、游戏进度、围棋对战、地图系统、角色成长、物品道具和任务系统。

| qi | integer | NOT NULL, DEFAULT 100 | 当前内力 |

| qi_regen_rate | integer | NOT NULL, DEFAULT 2 | 内力恢复速率（点/5分钟） |

**设计说明**：
- **体力消耗**：输掉一局围棋 -20体力，赢了不扣体力
- **内力消耗**：
  - 亢龙有悔：10内力
  - 独孤九剑：15内力
  - 腹语传音：20内力
  - 机关算尽：25内力
- **升级成长**：
  - 每升1级：max_stamina +10，max_qi +10
  - 经验值计算：next_level_exp = 100 * level * 1.5

## 四、核心游戏机制设计

### 1. 经验和等级系统

**经验获取**：
- 完成一局围棋（胜利）：+50 经验
- 完成一局围棋（失败）：+20 经验
- 完成主线任务：+100~500 经验
- 完成支线任务：+50~200 经验
- 使用技能：+5 技能经验

**等级计算**：
```typescript
function calculateExpToNextLevel(level: number): number {
  return Math.floor(100 * level * 1.5);
}

// 等级1→2: 150 EXP
// 等级2→3: 300 EXP
// 等级3→4: 450 EXP
// ...
```

**升级奖励**：
- max_stamina +10
- max_qi +10
- 技能点 +1（可用于升级技能）

async function checkQuestCompletion(userId: number, questId: string) {
  const progress = await getQuestProgress(userId, questId);
  const quest = await getQuest(questId);
  
  if (progress.progress.wins >= quest.requirements.wins_required) {
    // 完成任务
    await completeQuest(userId, questId);
    
    // 发放奖励
    await grantRewards(userId, quest.rewards);
    
    // 解锁技能
    if (quest.rewards.skills) {
      for (const skillId of quest.rewards.skills) {
        await unlockSkill(userId, skillId, questId);
      }
    }
  }
}

### 6. 技能解锁和升级机制

#### 技能解锁流程

1. **接受任务**：与NPC对话触发任务
2. **完成任务要求**：
   - 战胜指定NPC X次
   - 达到指定好感度
   - 完成前置任务
3. **获得技能**：
   - 插入 `player_skills` 记录
   - `unlocked = true`
   - `level = 1`
   - `unlocked_by_quest = quest_id`
4. **开始使用**：在围棋对局中可以使用该技能

**升级条件**：
- 使用技能累计达到一定次数

## 七、总结

### 新增表概览

本次设计新增了 **7张核心表**：

1. ✅ **player_stats** - 玩家属性（体力、内力、等级）
2. ✅ **player_skills** - 玩家技能（解锁、等级、使用统计）
3. ✅ **items** - 物品定义（药品、道具配置）
4. ✅ **player_inventory** - 玩家背包（持有物品）
5. ✅ **quest_progress** - 任务进度（详细追踪）
6. ✅ **npc_relationships** - NPC关系（好感度、交互）
7. ✅ **game_settings** - 游戏设置（偏好配置）

### 核心系统

- ⚡ **经验等级系统**：升级增加属性上限
- 💪 **体力系统**：输棋扣体力，限制对局频率
- 🔮 **内力系统**：技能消耗内力，策略使用
- 🎯 **任务系统**：完成任务解锁技能（如打败洪七公获得亢龙有悔）
- 💊 **物品药品**：恢复体力内力，提升战斗能力
- 📈 **技能成长**：技能可升级，效果更强、消耗更少

### 游戏平衡设计

- 输棋惩罚合理（-20体力），避免无限刷局
- 技能消耗内力，增加策略深度
- 自然恢复机制，鼓励适度游戏
- 任务门槛设计，形成成就感和目标导向
- 物品经济系统，提供货币消耗场景
