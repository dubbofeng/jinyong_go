## 🔧 使用示例

### 在组件中加载 NPC 数据

```typescript
import { loadNPC, loadDialogue } from '@/src/lib/data-loader';

// 加载 NPC 基础信息（优先数据库）
const npc = await loadNPC('hong_qigong');

// 加载对话数据（直接从 JSON）
const dialogue = loadDialogue('hong_qigong', 'zh');
```

### 在 API 路由中使用

```typescript
// app/api/npcs/[npcId]/route.ts
import { loadNPC, loadDialogue } from '@/src/lib/data-loader';

export async function GET(request, { params }) {
  const npc = await loadNPC(params.npcId);
  const dialogue = loadDialogue(params.npcId, 'zh');
  return Response.json({ ...npc, dialogue });
}
```

## ✅ 优势总结

### 开发体验
- ✅ **对话编辑**：直接修改 JSON，立即生效
- ✅ **版本控制**：Git 管理配置文件

- ✅ **性能优化**：静态内容编译时打包
- ✅ **灾难恢复**：JSON 种子快速恢复
