**使用流程**:
1. 在 `.env.local` 配置API密钥（三选一）
2. 访问 `http://localhost:9999/zh/admin/assets`
3. 选择分类或点击"批量生成全部"
4. 等待AI生成（自动缓存到 `public/generated/`）
5. 在游戏中引用：`/generated/{category}/{id}.png`

## 问题追踪

### 已修复的Bug ✅
1. 空格键导致页面滚动 → 添加 preventDefault()
2. 打字机效果state闭包问题 → 改用本地变量
3. 对话无法继续到下一节点 → 添加 dialogueUpdateTrigger 计数器
4. 游戏键盘与对话键盘冲突 → 使用 isDialogueVisibleRef 隔离
5. 对话框位置异常 → 使用 inline styles + fixed positioning
