# 完整开发计划文档

## 项目总体规划

**项目名称**：《金庸棋侠传》
**总体周期**：12-16周（三个版本迭代）
**技术栈**：Next.js 14 + RPG-JS v4 + KataGo WASM + PostgreSQL + Vercel
**目标用户**：围棋爱好者、武侠迷、策略游戏玩家

## 技术实现细节

### Google TTS对话语音集成方案

#### 技术选型
- **API**: Google Cloud Text-to-Speech API
- **语言**: 中文普通话（cmn-CN）
- **音色**: 多个Wavenet高质量音色（为不同NPC选择不同音色）
- **格式**: MP3（适合Web播放）
- **实现方式**: 服务端API + 前端播放器

#### 实现步骤

**1. Google Cloud配置**
```bash
# 安装Google Cloud SDK
npm install @google-cloud/text-to-speech

# 创建服务账号并下载密钥
# 在Google Cloud Console启用Text-to-Speech API
```

**2. 服务端API实现** (`app/api/tts/route.ts`)
```typescript
// 接收对话文本，返回音频URL
// 支持缓存机制（相同文本不重复生成）
// NPC音色映射表
```

**3. 音频缓存策略**
- 首次生成的音频保存到Vercel Blob Storage
- 使用对话文本hash作为文件名
- 数据库记录音频文件映射关系
- 减少API调用次数和成本

**4. 前端播放器组件** (`src/components/DialoguePlayer.tsx`)
```typescript
// 对话框组件集成音频播放
// 支持播放、暂停、跳过控制
// 显示播放进度和文字同步高亮
// 用户可选择纯文字/语音/同步模式
```

**5. NPC音色配置**
```json
{
  "洪七公": {
    "voice": "cmn-CN-Wavenet-C",
    "pitch": -2.0,
    "speakingRate": 0.9
  },
  "令狐冲": {
    "voice": "cmn-CN-Wavenet-B",
    "pitch": 0.0,
    "speakingRate": 1.0
  },
  "郭靖": {
    "voice": "cmn-CN-Wavenet-D",
    "pitch": -1.0,
    "speakingRate": 0.95
  }
}
```

#### 成本估算
- Google TTS定价：$4 / 100万字符（Wavenet音色）
- 预估对话总量：约5万字（V1.0）
- 预计成本：约$0.20（V1.0阶段）
- 免费额度：每月免费100万字符（Standard音色）

#### 优化建议
- 关键对话使用Wavenet高质量音色
- 普通对话使用Standard音色（免费额度）
- 预先生成所有对话音频（构建时）
- 打包到静态资源，无需运行时调用API
