# 🎬 动画效果快速测试

## 立即查看动画效果

### 方法 1：直接访问游戏

1. **确保服务器运行中**：
```bash
pnpm dev
```

2. **打开浏览器访问**：
```
http://localhost:9999/zh/game
```

3. **你会看到以下动画**：

#### 🚪 传送门脉冲动画
- **位置**：地图顶部中央（坐标 10, 1）
- **效果**：
  - 紫色光晕呼吸效果
  - 门emoji图标
  - 光晕不断扩散和收缩
  - 周期：1秒

#### 🔥 火焰动画瓦片
- **位置**：地图四个角落区域
  - 左上角：(3-5, 3)
  - 右上角：(14-16, 3)
  - 左下角：(3-5, 11)
  - 右下角：(13-15, 11)
- **效果**：
  - 红→橙→黄→橙循环闪烁
  - 快速切换（150ms/帧）
  - 模拟火焰跳动

## 动画系统位置

### 传送门脉冲动画
- **文件**: [src/lib/tilemap-engine.ts](../src/lib/tilemap-engine.ts)
- **方法**: `renderPortals()` (第 286 行)
- **核心代码**:
```typescript
const pulse = Math.sin(this.animationTime / 500) * 0.2 + 0.8;
// 使用正弦波实现0.8-1.0透明度循环
```

### 瓦片帧动画
- **文件**: [src/lib/tilemap-engine.ts](../src/lib/tilemap-engine.ts)
- **方法**: `renderTile()` (第 234 行)
- **核心代码**:
```typescript
const currentFrameIndex = Math.floor(this.animationTime / tileDef.animationSpeed) % frameCount;
// 根据时间计算当前应显示的帧
```

## 交互测试

### 测试传送门
1. 使用 WASD 或方向键移动玩家
2. 走到地图顶部中央的传送门位置
3. 按**空格键**或 **Enter**
4. 应该弹出确认对话框："是否Tavern？"
5. （目前多地图切换未实现，点确定不会有反应）

### 测试NPC对话
1. 移动到任一NPC附近
   - 洪七公：地图左侧中部 (5, 5)
   - 令狐冲：地图右侧中部 (15, 7)
   - 郭靖：地图中央下方 (10, 10)
2. 按**空格键**或 **Enter**
3. 对话框打开，查看多语言对话内容

## 预期结果

✅ **正常情况**：
- 传送门光晕流畅脉动
- 火焰瓦片快速闪烁
- 游戏整体流畅运行

1. 检查浏览器控制台是否有错误
2. 确认服务器正常运行

4. 清除浏览器缓存后重试
