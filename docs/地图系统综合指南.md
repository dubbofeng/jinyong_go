# 地图系统综合指南

**最后更新**：2026年1月27日

## 📋 系统概述

本项目采用 **Isometric（等距视图）** 地图系统，支持：
- ✅ 数据库驱动的地图管理
- ✅ 可视化地图编辑器
- ✅ 动态地图生成（Perlin Noise）
- ✅ 多种地形主题
- ✅ Autotile 自动瓦片系统
- ✅ 建筑和装饰物品
- ✅ 传送门系统

## 🗺️ 地图类型

## 🗄️ 数据库结构

### Maps 表
```sql
CREATE TABLE maps (
  id INTEGER PRIMARY KEY,
  map_id TEXT UNIQUE NOT NULL,
  name TEXT NOT NULL,
  description TEXT,
  map_type TEXT NOT NULL,  -- 'world' | 'scene'
  width INTEGER NOT NULL,
  height INTEGER NOT NULL,
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL
);
```

### MapTiles 表
```sql
CREATE TABLE map_tiles (
  id INTEGER PRIMARY KEY,
  map_id TEXT NOT NULL,
  x INTEGER NOT NULL,
  y INTEGER NOT NULL,
  tile_type TEXT NOT NULL,  -- 'grass', 'water', 'mountain', etc.
  FOREIGN KEY (map_id) REFERENCES maps(map_id)
);
```

### MapItems 表
```sql
CREATE TABLE map_items (
  id INTEGER PRIMARY KEY,
  map_id TEXT NOT NULL,
  item_name TEXT NOT NULL,
  item_path TEXT NOT NULL,
  item_type TEXT NOT NULL,  -- 'building', 'decoration', 'portal'
  x INTEGER NOT NULL,
  y INTEGER NOT NULL,
  width INTEGER,
  height INTEGER,
  metadata TEXT,  -- JSON: { targetMap, targetX, targetY } for portals
  FOREIGN KEY (map_id) REFERENCES maps(map_id)
);
```

- **地图名称**: 显示名称（例如：`新手村`）

3. 点击"生成地图"

5. 自动跳转到地图编辑器

**操作说明**:

## 🛠️ 技术实现

### Isometric 坐标转换

```typescript
// 网格坐标 → 屏幕坐标
function cartesianToIsometric(gridX: number, gridY: number) {
  const tileWidth = 128;
  const tileHeight = 64;
  const screenX = (gridX - gridY) * (tileWidth / 2);
  const screenY = (gridX + gridY) * (tileHeight / 2);
  return { screenX, screenY };
}

// 屏幕坐标 → 网格坐标
function isometricToCartesian(screenX: number, screenY: number) {
  const tileWidth = 128;
  const tileHeight = 64;
  const gridX = (screenX / (tileWidth / 2) + screenY / (tileHeight / 2)) / 2;
  const gridY = (screenY / (tileHeight / 2) - screenX / (tileWidth / 2)) / 2;
  return { gridX, gridY };
}
```

### Perlin Noise 地形生成

```typescript
import { Noise } from 'noisejs';

function generateTerrain(width: number, height: number, theme: string) {
  const noise = new Noise(Math.random());
  const tiles = [];
  
  for (let y = 0; y < height; y++) {
    for (let x = 0; x < width; x++) {
      // 使用多层噪声创建自然地形
      const value1 = noise.perlin2(x * 0.1, y * 0.1);
      const value2 = noise.perlin2(x * 0.05, y * 0.05) * 0.5;
      const combined = value1 + value2;
      
      const tileType = mapValueToTile(combined, theme);
      tiles.push({ x, y, tile_type: tileType });
    }
  }
  
  return tiles;
}

function mapValueToTile(value: number, theme: string): string {
  // 根据噪声值和主题选择瓦片类型
  if (theme === 'forest') {
    if (value < -0.3) return 'water';
    if (value < 0.2) return 'grass';
    if (value < 0.5) return 'grass_dense';
    return 'forest';
  }
  // ... 其他主题的映射
}
```

### Autotile 系统

Autotile 根据相邻瓦片自动选择正确的瓦片变体：

```typescript
function getAutotileIndex(map: TileMap, x: number, y: number): number {
  const n = hasSameTile(map, x, y - 1);  // 北
  const e = hasSameTile(map, x + 1, y);  // 东
  const s = hasSameTile(map, x, y + 1);  // 南
  const w = hasSameTile(map, x - 1, y);  // 西
  
  // 转换为 4 位二进制索引
  return (n ? 8 : 0) + (e ? 4 : 0) + (s ? 2 : 0) + (w ? 1 : 0);
}
```

## 🔧 API 接口

### 地图 CRUD

```typescript
// 创建地图
POST /api/maps
Body: { mapId, name, description, mapType, width, height }

// 获取地图列表
GET /api/maps

// 获取单个地图
GET /api/maps/:mapId

// 更新地图
PATCH /api/maps/:mapId
Body: { name, description }

// 删除地图
DELETE /api/maps/:mapId
```

### 地图生成

```typescript
// 生成新地图
POST /api/maps/generate
Body: {
  mapId: string,
  name: string,
  description?: string,
  mapType: 'world' | 'scene',
  width: number,
  height: number,
  theme: 'forest' | 'mountain' | 'village' | 'river' | 'desert' | 'temple'

```

### 瓦片操作

```typescript
// 获取地图瓦片
GET /api/maps/:mapId/tiles

// 批量更新瓦片
POST /api/maps/:mapId/tiles/batch
```


## 🎯 使用示例

### 创建新地图

```typescript
// 1. 访问地图生成器
window.location.href = '/zh/admin/maps/generate';

// 2. 填写表单
const formData = {
  mapId: 'village_newbie',
  name: '新手村',
  description: '适合新手玩家的村庄',
  mapType: 'scene',
  width: 32,
  height: 32,
  theme: 'village'
};

// 3. 提交生成
const response = await fetch('/api/maps/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(formData)
});

// 4. 自动跳转到编辑器
if (response.ok) {
  window.location.href = '/zh/admin/map-editor';
}
```

```typescript
// 1. 加载地图数据
```

## 🚀 最佳实践

### 地图设计原则

1. **尺寸选择**
   - 世界地图：64×64（展示全局）
   - 大型场景：48×48（主要门派）
   - 小型场景：32×32（房间、小院）

2. **主题搭配**
   - 山地主题 → 山门、武馆
   - 森林主题 → 野外、隐居地
   - 村庄主题 → 城镇、集市
   - 寺庙主题 → 寺院、道观

3. **传送门布局**
   - 世界地图：均匀分布各门派入口
   - 场景地图：边缘设置出口
   - 避免传送门相邻（至少间隔3格）

4. **性能优化**
   - 限制物品数量（< 100个/地图）
   - 使用 Autotile 减少精灵数量
   - 实施视口裁剪，只渲染可见区域

## 🐛 故障排除

### 常见问题

**Q: 地图生成失败**
- 检查地图ID是否重复
- 确认数据库连接正常
- 查看控制台错误信息

**Q: 瓦片显示不正确**
- 确认 Autotile 文件存在
- 检查瓦片类型拼写
- 清除浏览器缓存

**Q: 传送门不工作**
- 验证 metadata JSON 格式
- 确认目标地图存在
- 检查目标坐标在地图范围内

**Q: 编辑器性能差**
- 减小地图尺寸
- 降低缩放级别
- 关闭网格显示
