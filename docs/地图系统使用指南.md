# 地图系统使用指南

## 📋 概述

本项目支持两种地图系统：
- **Tilemap** - 传统正交视图（华山、少林寺、襄阳城）
- **Isometric** - 等距视图（世界地图 + 场景地图）

---

## 🗺️ Tilemap 地图系统

### 特点
- ✅ 正交视图（俯视角度）
- ✅ 多层渲染（地面、装饰、碰撞）
- ✅ 瓦片集管理
- ✅ 离屏 Canvas 缓存
- ✅ 传送门动画

### 渲染引擎

**TilemapEngine** (`src/lib/tilemap-engine.ts`)：

```typescript
class TilemapEngine {
  // 初始化地图
  loadMap(mapConfig: MapConfig): void
  
  // 渲染循环
  render(deltaTime: number): void
  
  // 碰撞检测
  isWalkable(x: number, y: number): boolean
  
  // 传送门检查
  checkPortal(x: number, y: number): Portal | null

```

### 性能优化

1. **离屏 Canvas 缓存**
   - 地面层预渲染到离屏 Canvas
   - 每帧只需复制，无需重新绘制
   - 性能提升 60%

2. **分层渲染**
   - 地面层（静态，缓存）
   - 装饰层（半静态，按需刷新）
   - 角色层（动态，每帧更新）

3. **视野裁剪**
   - 只渲染可见区域
   - 自动计算可见瓦片范围

## 🎲 Isometric 地图系统

### 特点
- ✅ 等距视图（钻石网格）
- ✅ 数据库驱动
- ✅ 动态生成和编辑
- ✅ 支持建筑、装饰、传送门
- ✅ 地图编辑器

### 数据库结构

```sql
-- 地图表
maps (id, map_id, name, map_type, width, height)

-- 瓦片表
map_tiles (id, map_id, x, y, tile_type)

-- 物品表
map_items (id, map_id, item_name, item_path, item_type, x, y, width, height)
```

### 地图类型

1. **世界地图**（64×64）
   - 展示各大门派位置
   - 传送门连接到场景地图

2. **场景地图**（32×32）
   - 华山场景
   - 少林寺场景
   - 武当山场景
   - 桃花岛场景

### 坐标转换

Isometric 坐标系统：

```typescript
// 网格坐标 → 屏幕坐标
function cartesianToIsometric(gridX: number, gridY: number) {
  const screenX = (gridX - gridY) * (tileWidth / 2);
  const screenY = (gridX + gridY) * (tileHeight / 2);
  return { screenX, screenY };
}

// 屏幕坐标 → 网格坐标
function isometricToCartesian(screenX: number, screenY: number) {
  const gridX = (screenX / (tileWidth / 2) + screenY / (tileHeight / 2)) / 2;
  const gridY = (screenY / (tileHeight / 2) - screenX / (tileWidth / 2)) / 2;
  return { gridX, gridY };
}
```

### 地图生成

使用 Perlin Noise 算法生成自然地形：

```typescript
import { Noise } from 'noisejs';

function generateIsometricMap(width: number, height: number, theme: string) {
  const noise = new Noise(Math.random());
  const tiles = [];
  
  for (let y = 0; y < height; y++) {
    for (let x = 0; x < width; x++) {
      const value = noise.perlin2(x * 0.1, y * 0.1);
      const tileType = mapValueToTile(value, theme);
      tiles.push({ x, y, tileType });
    }
  }
  
  return tiles;
}
```

### 4种地形主题

1. **山地**（mountain）
   - 高海拔地形
   - 岩石和草地
   - 稀疏植被

2. **森林**（forest）
   - 茂密树木
   - 绿色为主
   - 水源点缀

3. **村庄**（village）
   - 平坦地形
   - 适合建筑
   - 道路网络

4. **河流**（river）
   - 蜿蜒水系
   - 沿岸植被
   - 桥梁和渡口

## 🎨 添加地图资源

### 1. 瓦片资源

放置在 `public/game/isometric/autotiles/`：

```

├── water.png       # 水面瓦片
├── wood.png        # 木地板

**瓦片规格**：
- 尺寸：128×64 像素

- 命名：小写，下划线分隔

放置在 `public/game/isometric/buildings/`：

```
