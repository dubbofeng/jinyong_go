# 环境配置说明

## 数据库配置

### 1. 创建Vercel Postgres数据库
访问 [Vercel Postgres](https://vercel.com/storage/postgres) 创建数据库

### 2. 配置环境变量
将以下环境变量添加到 `.env.local` 文件：

```env
# Postgres Database URLs (从Vercel获取)
POSTGRES_URL=postgres://user:password@host/database
POSTGRES_PRISMA_URL=postgres://user:password@host/database?pgbouncer=true&connect_timeout=15
POSTGRES_URL_NON_POOLING=postgres://user:password@host/database

# NextAuth Secret (生成: https://generate-secret.vercel.app/32)
AUTH_SECRET=your-32-character-secret-here

# NextAuth URL (本地开发)
NEXTAUTH_URL=http://localhost:3000
```

### 3. 数据库迁移
```bash
# 生成迁移文件
pnpm drizzle-kit generate:pg

# 执行迁移
pnpm drizzle-kit push:pg
```

## 开发环境启动

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

访问 http://localhost:3000

## 生产环境部署

### Vercel部署
```bash
# 推送代码到GitHub
git push origin main

# 在Vercel控制台导入项目并配置环境变量
```

确保在Vercel项目设置中配置所有必要的环境变量。
