# KataGo WebAssembly 目录

此目录用于存放KataGo的WebAssembly文件和模型：
- katago.wasm - KataGo WebAssembly核心文件
- katago.js - KataGo JavaScript接口
- model.bin.gz - 轻量级围棋AI模型（9路/13路）

## 下载地址
- KataGo WASM: https://github.com/lightvector/KataGo
- 模型文件: 需要选择适合Web的轻量级模型

## 注意事项
- 模型文件较大，建议使用CDN或压缩
- 确保WASM文件正确配置CORS头
